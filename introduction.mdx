## Overview

Relevize empowers you to centralize all your project documentation in one intuitive platform. You create, collaborate, and publish docs without the hassle of scattered files or complex tools. Built for developers, product teams, and technical writers, Relevize handles version control, real-time collaboration, and powerful search so you focus on content.

Whether you document APIs, user guides, or internal wikis, Relevize scales with your needs. Access everything via a clean web interface or integrate with your CI/CD pipeline for automated updates.

<Callout kind="info">
  Relevize supports unlimited projects on all plans, with seamless Markdown and MDX editing.
</Callout>

## Get Started

Launch your first project in minutes. Follow these steps to set up Relevize.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Visit relevize.com and sign up with your email or GitHub account. Verify your email to unlock the dashboard.
  </Step>
  <Step title="New Project" icon="plus">
    Click "New Project" and name it, like "My API Docs". Add a description and choose visibility (public or private).
  </Step>
  <Step title="Add Content" icon="edit-3">
    Create your first page using the Markdown editor. Import existing docs from GitHub repos.
  </Step>
  <Step title="Publish" icon="globe">
    Preview changes, then publish. Share the live URL or connect a custom domain.
  </Step>
</Steps>

For programmatic access, use the Relevize API:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.relevize.com/v1/projects', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'My Project',
      description: 'API documentation'
    })
  });
  ```
  ```python
  import requests

  headers = {
      'Authorization': f'Bearer {YOUR_API_KEY}',
      'Content-Type': 'application/json'
  }
  data = {
      'name': 'My Project',
      'description': 'API documentation'
  }
  response = requests.post('https://api.relevize.com/v1/projects', json=data, headers=headers)
  ```
</CodeGroup>
